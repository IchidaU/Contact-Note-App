## プロジェクトの概要

2025年10月頃に、架空の中学校での連絡帳をシステム化する、という想定で制作した成果物です。
作業期間は4週間、100時間以内という制限があり、提示された要求に従い、要件定義・設計・開発を行いました。

## 今後の展望・改善点

本システムの運用と発展の面から、以下の改善点や新機能の実装を検討しています。

- 現在UI上からは行えない、ユーザー自身によるパスワードリセット機能を実装する。
- 現在の新規ユーザー作成機能に加え、進級・卒業・異動等に伴うステータス変更やアカウント削除機能を実装し、より実践的なシステムにする。
- 学年主任の機能に、現在の「気になる」生徒の個別状況把握に加えて、学年全体やクラス別の状況・サマリーをグラフ等で把握できるダッシュボードを実装する。
- 担任の個別メモとは別に、学年全体やクラス単位の運用に関して主任が書き込めるメモ機能を追加する。
